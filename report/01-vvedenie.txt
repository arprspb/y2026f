Введение

Цель работы — реализовать автоматизированную систему транскрибации голосовых команд оператора с использованием open-source ASR (VOSK), сохранением аудио и результатов в базе данных, веб-интерфейсом с прослушиванием записей, а также аутентификацией, ролями и сценариями из задания (запись команды, проверка и подтверждение, поиск по истории, администрирование пользователей).

Краткое описание решения. Клиентская часть на Vue 3 и Vite отображает экран входа, запись с микрофона с предпросмотром распознавания, историю с фильтрами, карточку записи с аудио и правками, страницу пользователей для администратора. Серверная часть на FastAPI принимает загрузку аудио, конвертирует в формат WAV 16 кГц моно через ffmpeg, распознаёт речь моделью VOSK из каталога, задаваемого переменной окружения; из текста извлекаются ключевая фраза команды из фиксированного набора и идентификатор (цифры, буквы или смесь). Данные хранятся в PostgreSQL (SQLAlchemy 2, миграции Alembic). Первичный администратор создаётся при старте из переменных окружения; саморегистрация отключена. Роли: администратор, оператор записи, оператор проверки; доступ к операциям разграничен на уровне API.
